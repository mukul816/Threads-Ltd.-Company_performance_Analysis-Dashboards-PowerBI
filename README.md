# Threads-Ltd.-Company_performance_Analysis-Dashboards-PowerBI
# Threads Ltd. Performance Analysis Dashboard (Power BI)
# Overview

This project presents a Power BI dashboard created while learning on DataCamp. It analyzes the performance of Threads Ltd. and provides insights for top management (CCO and business heads).

# Objective
Analyze company performance

Track key KPIs like revenue, profit, and sales

Support executive-level decision making
